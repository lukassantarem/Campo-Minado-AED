# Campo Minado - AED

Projeto acadêmico desenvolvido para a disciplina de Algoritmos e Estruturas de Dados.

## Descrição

Implementação modular do jogo Campo Minado em linguagem C, com diferentes níveis de dificuldade e gerenciamento dinâmico de matrizes.

O projeto evoluiu a partir de uma versão inicial e incorporou novas funcionalidades, incluindo sistema de logs em arquivo texto, uso de Tipos Abstratos de Dados (TAD), recursão para algoritmo Flood Fill e automação de testes via arquivos de entrada.

## Funcionalidades

* Múltiplos níveis de dificuldade
* Posicionamento aleatório de minas
* Cálculo automático de minas adjacentes
* Algoritmo Flood Fill recursivo
* Registro completo da partida em `log.txt`
* Leitura automatizada de jogadas por arquivo de teste
* Verificação de vitória e derrota

## Tecnologias

* C

## Conceitos aplicados

* TAD (Tipo Abstrato de Dados)
* Recursão
* Manipulação de arquivos
* Alocação dinâmica de memória
* Matrizes dinâmicas
* Modularização de código
* Testes automatizados
